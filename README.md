# 📚 GRE3000 Vocab App

一个本地运行的 GRE 词汇背诵网页应用。它把 GRE 词表、发音、选择题复习、错题统计和词典搜索放在同一个网页里，适合在电脑或手机浏览器中进行日常背词。

![主页截图](docs/screenshots/home.png)

## ✨ 这个应用可以做什么

- 按随机顺序背诵全部 3041 个 GRE 词汇。
- 每 20 个词为一组，按组推进学习。
- 阶段一用于熟悉单词、中文释义、英文释义和发音。
- 阶段二用四选一选择题检查中文释义。
- 阶段三复习已经通过阶段二的单词。
- 答错会自动记录错误次数。
- 可以按错误次数筛选单词，也可以按错误次数倒序复习。
- 可以在主页直接搜索单词并查看释义和发音。
- 所有学习进度保存在当前浏览器中，不需要账号。

## 🏠 主页怎么用

主页包含四个主要区域：

- **总进度**：查看总词数、已通过阶段二的词数、已完成组数。
- **阶段一 / 阶段二**：开始新一组学习，或重新背诵指定组。
- **阶段三**：复习已经通过阶段二的词。
- **词典 / 搜索**：搜索单词、中文释义或英文释义。

![主页截图](docs/screenshots/home.png)

常用操作：

1. 点击 **继续下一组**，从还没完成的组开始学习。
2. 在组选择框中选择某一组，再点击 **重背所选组**，可以重新背这一组。
3. 在搜索框输入单词或释义，可以快速查词。
4. 点击 **打开完整词典**，可以进入详细查询页面。

## 🃏 阶段一怎么用

阶段一是记忆阶段。页面左侧显示单词、音标和发音按钮，右侧直接显示中文释义。这样可以把单词词形和中文含义放在同一个视觉区域里，更容易形成记忆绑定。

![阶段一截图](docs/screenshots/stage-one.png)

操作步骤：

1. 进入单词时，应用会自动播放一次发音。
2. 看左侧单词和右侧中文释义。
3. 需要重复听发音时，点击喇叭按钮。
4. 点击 **下一个** 进入下一词。
5. 背完当前组后，会自动进入阶段二。

## ✅ 阶段二怎么用

阶段二是中文释义选择题。页面会显示当前单词和 4 个中文释义选项，其中只有 1 个是正确答案。

![阶段二截图](docs/screenshots/stage-two.png)

操作步骤：

1. 进入每个新词时，应用会自动播放一次发音。
2. 根据单词选择正确的中文释义。
3. 选对后会自动进入下一个词。
4. 选错后会显示正确中文释义和英文释义，并把这个词的错误次数加 1。
5. 当前组结束后，会自动重测本组错词。
6. 直到本组全部正确，才会进入下一组阶段一。

顶部的 **上一个单词** 区域可以回看上一词的中英文释义，也可以点击小喇叭重新播放上一词发音。

## 🔁 阶段三怎么用

阶段三用于复习已经通过阶段二的词。

入口在主页的 **阶段三** 区域：

- **背诵全部已通过词**：复习所有已经通过阶段二的单词。
- **按错误数倒序**：优先复习错得最多的词。
- **错误数等于某个数值**：只复习错误次数等于指定数字的词。

阶段三的做题方式和阶段二一致：

1. 看单词并听发音。
2. 从 4 个中文释义中选择正确答案。
3. 答错会增加错误次数。
4. 一组复习完后会继续进入阶段三下一组。

## 🔍 词典搜索怎么用

主页可以直接搜索单词、中文释义或英文释义。点击搜索结果或 **打开完整词典** 后，会进入详细词典页面。

![词典截图](docs/screenshots/dictionary.png)

词典详情页会显示：

- 单词
- 发音按钮
- UK / US 音标
- 中文释义
- 英文释义
- 所在组
- 全局序号
- 当前错误次数

## 💾 学习进度保存在哪里

学习进度保存在当前浏览器的 `localStorage` 中。

这意味着：

- 关闭网页后再打开，进度还在。
- 换浏览器或换设备，进度不会自动同步。
- 清除浏览器数据后，进度会丢失。
- 这个应用不需要账号，也不会上传你的学习数据。

## 🖥️ 本地运行方式

如果你是从源码运行这个应用，先安装依赖：

```bash
npm install
```

启动本地网页：

```bash
npm run dev
```

默认访问地址：

```text
http://127.0.0.1:5173/
```

## 🛠️ 重新生成词表和音频数据

如果源 Excel 或音频文件有更新，可以重新生成静态数据：

```bash
npm run generate:data
```

默认源文件位置：

```text
D:\桌面\LGU\GRE\3000-dev\3000.xlsx
D:\桌面\LGU\GRE\3000-dev\3000_wav
```

生成后的文件：

- `src/data/vocab.ts`
- `public/audio/*.wav`

## ⚠️ 注意事项

- 本应用是本地网页版本，不包含账号、云同步或后端服务。
- 进度只保存在当前浏览器中。
- 如果浏览器阻止自动播放声音，请先手动点击一次页面上的发音按钮。
- 如果公开分享项目，请确认词表和音频素材有公开分享权限。
